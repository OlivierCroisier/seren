SEREN


Meet SEREN, the Java SERialization ENhancer !

Seren optimizes your Java classes so that they are much quicker to serialize.
Easy to use, risk-free, performant and customizable, try Seren and improve the performance of your application in minutes !

Easy to use

  1. Download the Seren archive
  2. Put its configuration file in the classpath
  3. Use a JVM argument to load the library
  4. ... profit !

Risk-free

Seren instruments your classes at load-time, so you don't have to change anything in your code or configuration.

Easy to install, easy to remove, try Seren now at no risk !

Performant

Seren can speedup your serialization process by a 1X - 3X factor, depending on your objects' structure.

Strings and numeric wrappers get the best performance improvement.

Customizable

Seren uses cumstomizable filters to decide which classes to enhance. A few filters are provided out-of-the-box : by class name, by package, using annotations... Or you can easily write your own !
