How it works

Seren is a Java Agent. This is a standard Java mechanism (since Java 5.0), that allows third-party libraries to instrument class when they are loaded into the JVM.

Seren is no black magic : it just plugs into the standard Java serialization system by generating optimized readObject / writeObject methods.

Final fields

As per the Java Language Specification, final fields must be set to some value before the object's instanciation process is over, that is, in the constructor. Then, a freeze occurs on the value and it cannot be changed anymore. The compiler enforces this
